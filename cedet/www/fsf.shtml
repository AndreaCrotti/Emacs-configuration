About free software I have written

In my anonymous FTP top level directory you can currently download several programs I've written.

GNU talk The latest version of GNU Talk, a BSD compatible unix talk program with interfaces in curses, Xt/Athena/Lesstif, Gtk/Gnome, and Emacs.

Speedbar An Emacs Lisp program which allows you to create a special skinny frame with a specialized directory listing in it.

The Semantic Bovinator An Emacs Lisp program which is a parser generator for Emacs Lisp parsers.

Checkdoc An Emacs Lisp program which checks the style of your documentation strings and comments. Useful if you want to keep other Emacs Lisp gurus from picking on you.

C-Parse An Emacs Lisp program which can parse a c file, and allows searching for functions, variables, and types.

EDE Emacs Development Environment implements projects under emacs making it easy to maintain programs without learning make.

EIEIO: Enhanced Implementation of Emacs Interpreted Objects is a package which implements a CLOS subset for Emacs. It also includes examples which can draw simple tree graphs, bar charts.

Working library. Display eye candy while waiting for emacs to do something.

X11 lib calls for emacs, V 0.3a Imagine the binary network interface for X windows implemented in emacs lisp. Is it useful? Silly? I dunno, but it was fun to play with. No documentation. Byte compile it, load "xhello.el" and run the function `XX' for the simple demo.

Hangman game for emacs. About as simple as it gets.
